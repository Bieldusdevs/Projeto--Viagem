# ✈️ SkyFly Travel - Flight Booking System

SkyFly is a high-end, modern flight ticket sales website built with a cutting-edge tech stack. It focuses on user experience, stunning visuals with 3D integration, and a seamless booking flow.

## 🚀 Tech Stack

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [Three.js](https://threejs.org/) via [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) & [Drei](https://github.com/pmndrs/drei)

## ✨ Features

- **Immersive 3D Hero**: Interactive 3D plane and atmosphere using Three.js.
- **Dynamic Search**: Fully responsive search form with smooth transitions.
- **Flight Filtering**: Filter flights by class (Economy, Executive, First Class).
- **Complete Checkout Flow**: From selection to passenger data and payment simulation.
- **Digital Ticket**: A beautifully designed digital boarding pass upon confirmation.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile.

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/skyfly-travel.git
   cd skyfly-travel
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open the browser**:
   Visit `http://localhost:3000` to see the site in action.

## 📁 Project Structure

- `/app`: Next.js App Router pages and layouts.
- `/components/ui`: Reusable atomic UI components (shadcn pattern).
- `/components/canvas`: Three.js scenes and 3D objects.
- `/lib`: Utility functions and helpers.
- `/public`: Static assets and images.
